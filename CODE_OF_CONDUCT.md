# Code of Conduct

This project adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating, you agree to uphold it.

## Reporting

Report unacceptable behavior to **iiitmahesh@gmail.com**. All complaints will be reviewed and
investigated promptly and fairly. Project maintainers are obligated to respect the privacy and
security of anyone reporting an incident.
